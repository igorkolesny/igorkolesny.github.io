Версия 0.1 Альфа (от 08.09.2020):
- Релиз альфа версии
Альфа версия содержит пакеты для загрузки такие как: Браузеры, Кодеки, Библиотеки.

Список браузеров содержит:

  • - Mozilla Firefox
  • - Google Chrome
  • - Opera
  • - Opera GX
  • - Yandex Browser
  • - Microsoft Edge (новый, на движке Chromium)


Список кодеков содержит:

  • - K-Lite Codec Pack Standart
  • - XviD Codec
  • - Apple Quick Time


Список библиотек содержит:

  • - DirectX
  • - .NET Framework 4.5
  • - .NET Framework 4.7
  • - nVidia PhysX
  • - VC Redist 2015, 2017 и 2019 (x64, x86)
  • - VC Redist 2013 (x64, x86)


Изменения в функционале:

  • - Добавлен выбор каталога для загрузки. Выбрать можно по нажатию на соответствующую кнопку или ввести путь для сохранения вручную
  • - Можно проверить сервера загрузки (для сомневающихся пользователей). Отображаются все ссылки на источники загрузки. Ссылки можно скопировать и проверить вручную через браузер.
